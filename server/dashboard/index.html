Installation Steps

1
Download the theme ZIP from above
2
Upload to Shopify → Online Store → Themes → Add theme → Upload ZIP
3
Create a license in the Licenses tab for the store's domain
4
Open Theme Settings → License & Protection and enter the license key + server URL
5
Publish the theme and visit the store to verify it loads

Architecture

Shell The theme installed on Shopify. Contains empty section containers with no CSS or HTML content.
Brain This server. Validates licenses and renders all section HTML, CSS, and JavaScript on every page load.
Loader A script that reads shell sections, sends data to this server, and injects the rendered content back.
Kill Switch If the license is invalid, all content is hidden and a license error is shown. Cannot be bypassed.
