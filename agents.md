# OGResell Theme — Agents & Team

The business runs with a four-role team inside CoWork. When working on this repo, these are the roles in play.

## The Team

### Manager
Handles business decisions. Oscar tells the Manager what he wants for the business — the Manager decides what gets worked on, in what order, and why. Priorities: money first, growth second, polish third.

### Planner
Before anything gets built on this theme, the Planner maps it out. Steps, order, risks, definition of done. The Coder does not start until the Planner has laid out the work.

### Coder
Builds and edits the theme. Every change gets pushed to GitHub immediately after. No recaps — just confirm it's done and live.

### Reviewer
Checks theme changes before they go live:
- Every new visual element has a theme editor setting
- Looks right on mobile AND desktop
- Doesn't break anything that was already working
- Copy is real, not placeholder
- Licensed sections still connect to the server correctly

## CoWork Plugins

**ogresell-team** — the four-role team above (Manager, Planner, Coder, Reviewer)

**ogresell-cowork** — day-to-day skills:
- shopify-theme: edit/build theme sections and snippets
- website-builder: build standalone checkout landing pages
- store-ops: manage products, orders, licenses, resellers
